# Git使用说明

## 添加github或gitee账号

> 要开启 `梯子` ，通过 Token 方式添加

![image-20260530133609846](./IDEA.assets/image-20260530133609846.png)

> 如果添加不成功，则尝试先将 `HTTP Proxy` 设置为 `No proxy（不使用任何代理，直接进行网络连接）` ，等添加成功后再设置回 `Auto-detect proxy settings（自动读取并使用操作系统的代理配置）`
>
